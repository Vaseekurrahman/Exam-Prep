# ExamPrep 🎓

ExamPrep is an online test preparation platform built using the **MERN stack** (MongoDB, Express.js, React, Node.js).  
It allows students to practice quizzes, track their progress, and improve exam performance.

---

## 🚀 Features
- 🔑 User Authentication (Login/Signup)
- 📝 Create and attempt quizzes
- 📊 Track performance and scores
- 📚 Question bank management
- 🎨 Responsive UI with React

---

## 🛠️ Tech Stack
- **Frontend:** React, Redux, Bootstrap/Tailwind
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens), bcrypt

---

## ⚙️ Installation and Setup
Clone the repository and run locally.
